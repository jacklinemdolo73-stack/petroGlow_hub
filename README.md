# PetroGlow Fuel Ordering System

## Description

A simple PHP and MySQL web application for processing fuel orders. The system validates stock availability, calculates order costs, updates inventory, and generates a receipt for successful orders.

## Features

* Select fuel products
* Order fuel by liters
* Stock availability validation
* Automatic total cost calculation
* Inventory update after purchase
* Receipt generation

## Technologies Used

* PHP
* MySQL
* HTML
* CSS

## Installation

1. Install XAMPP/WAMP.
2. Create a MySQL database.
3. Create the `products` table.
4. Configure database credentials in `db.php`.
5. Place project files in the web server directory.
6. Start Apache and MySQL.
7. Open the application in a browser.



## How It Works

1. User selects a product and quantity.
2. System checks available stock.
3. Total cost is calculated.
4. Stock is updated in the database.
5. Receipt is displayed to the user.
